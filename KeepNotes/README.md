# KeepNotes

KeepNotes is a simple web-based note-taking application built with **Node.js**, **Express**, and **EJS**. It allows users to create, read, edit, and delete notes in a clean and organized interface.

## Live Demo

Check out the live app here: [https://keepnotes-dqzz.onrender.com](https://keepnotes-dqzz.onrender.com)

## Features

- Create notes with a **title** and **description**.
- View all notes on the home page.
- Read the full content of a note.
- Edit note titles.
- Delete notes.
- Files are stored locally in the `files/` folder.

## Tech Stack

- **Backend:** Node.js, Express
- **Frontend:** EJS templates, TailwindCSS
- **File Storage:** Local file system

## Usage

Go to the home page to see all notes.

Click Read More to view a note.

Click Edit filename to rename a note.

Click Delete file to remove a note.
